**Responsive Website using HTML, CSS, JS, and Bootstrap:**

This project is a fully responsive website built with HTML, CSS, and Bootstrap. The website adapts seamlessly to different screen sizes, providing a user-friendly experience across all devices, from mobile phones to desktops. It features a modern, clean layout, utilizing Bootstrap for responsive design and styling.

Key Features:
Responsive Design: The website automatically adjusts to fit all screen sizes, ensuring usability on mobile, tablet, and desktop devices.

Clean Layout: A simple, modern design with a focus on ease of navigation.

Bootstrap Integration: Using Bootstrap to create a responsive, mobile-first layout.

**Technologies Used:**

* HTML: For the structure and content of the website.

* CSS: For custom styling and design enhancements.

* Bootstrap: For building a responsive, grid-based layout and UI components.
* JavaScript: This is used to create simple type animation in the Project.
